# LA DOCE — Sitio Web Oficial 🏆

Sitio web oficial de **LA DOCE**, el bazar deportivo líder en Pucallpa, Perú.  
Desarrollado por **Kemper Aspajo**.

---

## 🚀 Tecnologías

| Tecnología | Uso |
|---|---|
| HTML5 | Estructura de páginas |
| CSS3 (Vanilla) | Estilos y animaciones |
| JavaScript (ES6+) | Interactividad y lógica |
| [EmailJS](https://www.emailjs.com/) | Envío de formulario de empleo |

---

## 📁 Estructura del proyecto

```
La-Doce-Website/
├── index.html              # Página principal
├── trabaja.html            # Página "Trabaja con nosotros"
├── static/
│   ├── css/
│   │   ├── style.css       # Estilos globales + dark/light mode
│   │   └── trabaja.css     # Estilos de la página de empleo
│   ├── js/
│   │   ├── main.js         # Theme toggle + scroll reveal (index)
│   │   └── trabaja.js      # Theme toggle + formulario EmailJS
│   └── images/             # Imágenes y logotipo
└── README.md
```

---

## 🌐 Deploy

El sitio está pensado para desplegarse como sitio estático en:
- **GitHub Pages** (gratis)
- **Vercel** (próximamente)

No requiere backend ni base de datos.

---

## 📬 Formulario de empleo

El formulario de "Trabaja con nosotros" usa **EmailJS** para enviar las solicitudes directamente al correo del negocio sin necesidad de servidor.
